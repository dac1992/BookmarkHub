## 技术栈

1. **框架选择**
   - React 18
   - TypeScript 5
   - TailwindCSS

2. **工具库**
   - React Icons
   - React Query
   - Framer Motion

3. **构建工具**
   - Vite
   - PostCSS
   - ESLint

## 开发规范

1. **代码规范**
   - 使用 TypeScript
   - 遵循 ESLint 规则
   - 编写单元测试

2. **样式规范**
   - 使用 TailwindCSS
   - 遵循响应式设计
   - 支持深色模式

3. **组件规范**
   - 函数组件
   - Hooks 开发
   - Props 类型定义
